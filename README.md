# Naive-Wi-Fi-Sample
A sample for using [Native Wi-Fi](https://docs.microsoft.com/en-us/windows/desktop/nativewifi/portal) programming.

## Description
* Show available network adapters.
* Show available networks.
* Connect a specific network.
